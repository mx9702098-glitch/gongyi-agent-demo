# 公益 Agent 对话 Demo

微信 Agent 平台公益 Skill 的对话式交互原型。

## 在线预览

打开 [index.html](./index.html) 或访问 GitHub Pages 链接（部署后自动生成）。

## 设计目标

- 把腾讯公益能力原子化，接入微信大 Agent
- 在对话上下文里让用户「顺手做好事」，不做电商式的 SKU 切换
- 卡片 → 半屏 → 小程序 三层递进，遵循微信 Agent 交互规范

## 包含路径

- P1 模糊意图：意图收窄 → 推荐卡 → 深度卡 → 捐助半屏 → 完成卡
- P2 清晰意图：直达推荐卡
- P3 老用户：查进展 / 再捐 / 看花园 三条回路

## 技术

- 单文件 HTML，无外部依赖（图片来自 Unsplash CDN）
- 纯 CSS + 原生 JS
